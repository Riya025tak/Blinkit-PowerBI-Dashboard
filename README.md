Project Title:
Blinkit Sales & Operations Performance Dashboard – Power BI


Project Purpose / Description:
Developed an interactive Power BI dashboard to analyze Blinkit’s sales performance, customer behavior, and operational efficiency. The project aims to provide data-driven insights to help stakeholders track key KPIs, identify trends, and support strategic decision-making in a quick-commerce (Q-commerce) environment.


Tech Stack:
1. Power BI (DAX, Power Query)
2. MS Excel (data cleaning & preprocessing)
3. SQL (for data understanding & structuring)


Data Source:
Publicly available / simulated Blinkit sales dataset
Data includes orders, revenue, product categories, delivery metrics, customer segments, and time-based data



Features & Analysis:
1. Sales performance analysis by category, region, and time period
2. Revenue, order volume, average order value (AOV) KPIs
3. Customer segmentation and purchase behavior insights
4. Time-series analysis to identify peak demand hours and seasonal trends
5. Delivery and operational metrics to evaluate efficiency


Key Highlights / Outcomes:
1. Built dynamic dashboards with slicers, drill-downs, and filters for easy stakeholder interaction
2. Applied DAX measures for advanced KPI calculations and trend analysis
4. Enabled quick insights into high-performing products and underperforming categories
5. Demonstrated strong business storytelling and visualization best practices
6. Showcased how analytics can support inventory planning and operational optimization
